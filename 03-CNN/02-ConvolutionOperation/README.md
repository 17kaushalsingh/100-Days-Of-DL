# Convolution Operations
- Point wise product of input matrix and kernel/filter and summing them up
- `Greyscale Images:` Matrix(M, M) conv. Filter (N, N) => Feature Map(M-N+1, M-N+1)
- `RGB Images:` Matrix(M, M, 3) conv. Filter (N, N, 3) => Feature Map(M-N+1, M-N+1)
